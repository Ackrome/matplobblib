6. Основные библиотеки для работы с изображениями: OpenCV, Pillow, matplotlib.

Для работы с компьютерным зрением на языке Python есть основные библиотеки, каждая из которых решает свой круг задач: от математической обработки до визуализации.1. OpenCV (Open Source Computer Vision Library) — самая мощная и популярная библиотека для профессионального компьютерного зрения. Она написана на C++, поэтому работает очень быстро и позволяет обрабатывать видео в реальном времени. Её главная задача — реализация сложных алгоритмов (фильтрация, поиск объектов, работа с камерой).Основные функции: cv2.imread() (чтение), cv2.cvtColor() (смена цветовой модели), cv2.GaussianBlur() (размытие), cv2.Canny() (поиск границ).Пример кода:img = cv2.imread('image.jpg')gray = cv2.cvtColor(img, cv2.COLOR\_BGR2GRAY) # Перевод в ч/бP.S. OpenCV по умолчанию читает изображения в формате BGR, а не RGB.2. Pillow (PIL — Python Imaging Library) — более легкая библиотека. Она идеально подходит для базовых манипуляций с изображениями, когда не нужны сложные математические алгоритмы. Её часто используют для предварительной подготовки датасетов.Основные функции: Image.open() (открытие), img.resize() (изменение размера), img.rotate() (поворот), img.filter() (базовые фильтры).Пример кода:from PIL import Imageimg = Image.open('pic.png').resize((224, 224)) # Открыть и сжать3. Matplotlib — это библиотека для построения графиков и визуализации данных. В компьютерном зрении она практически не используется для самой обработки, но незаменима для того, чтобы вывести результат работы алгоритма на экран, показать гистограммы или сравнить несколько картинок в одном окне.Основные функции: plt.imshow() (подготовка изображения к показу), plt.show() (вывод окна на экран), plt.subplot() (создание сетки из нескольких картинок).Пример кода:import matplotlib.pyplot as pltplt.imshow(img)plt.show() # Отрисовка изображения в интерфейсе

Краткий итог: OpenCV — для сложных алгоритмов и видео, Pillow — для быстрой подготовки и простых правок, Matplotlib — для визуализации.
