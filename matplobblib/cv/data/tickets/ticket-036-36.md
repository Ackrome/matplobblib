## 36. Задача сегментации изображений и её виды.

Задача сегментации изображений

0. Задача компьютерного зрения, заключающаяся в классификации каждого пикселя изображения с присвоением ему метки класса или идентификатора объекта.

1. Выходом алгоритма является маска.

2. Виды сегментации выделяются на основе отношения алгоритма к разделению объектов одного класса на независимые сущности и обработке фона: семантическая сегментация, сущностная сегментация, паноптическая сегментация.

#### Семантическая сегментация ⇔ Semantic Segmentation

0. Задача попиксельной классификации изображения, при которой каждому пикселю присваивается метка класса без разделения объектов одного класса на независимые экземпляры.

1. Объекты одного класса образуют на маске единое множество пикселей с одинаковым значением.

2. Алгоритм классифицирует все пиксели изображения, включая фон.

3. Вход: тензор размерности $$3 \\times H \\times W$$, где $$H$$ — высота, $$W$$ — ширина, 3 — количество цветовых каналов RGB.

4. Выход: тензор размерности $$C \\times H \\times W$$, где $$C$$ — количество классов. Каждому пикселю соответствует вектор оценок (scores) принадлежности к каждому из $$C$$ классов.

5. Финальная предсказанная маска размерности $$H \\times W$$ вычисляется посредством применения математической операции argmax по измерению $$C$$ для каждого пикселя.

#### Сущностная сегментация ⇔ Instance Segmentation

0. Задача одновременного обнаружения объектов и их попиксельной классификации с присвоением каждому экземпляру одного класса уникального идентификатора.

1. Алгоритм разделяет пересекающиеся или стоящие рядом объекты одного класса.

2. Выходом является набор BBox, меток классов и индивидуальных бинарных масок для каждого обнаруженного объекта.

3. Пиксели фона игнорируются алгоритмом и не получают меток классов.

#### Паноптическая сегментация ⇔ Panoptic Segmentation

0. Задача, объединяющая принципы семантической и сущностной сегментации.

1. Каждый пиксель изображения классифицируется и получает метку класса (включая фон).

2. Пиксели, принадлежащие к исчисляемым объектам, дополнительно получают уникальный идентификатор экземпляра для их разделения.
