22. Проблемы обучения глубоких нейронных сетей и методы их решения.

#### 1. Затухание и взрыв градиента

При обратном распространении ошибки градиенты проходят через много слоев. Если они многократно умножаются на малые значения, возникает затухание градиента: ранние слои почти не обучаются. Если значения слишком большие, возникает взрыв градиента: веса начинают резко изменяться, обучение становится нестабильным.

Методы решения:

- инициализация весов: Xavier (стабилизирует дисперсию при проходе через слои – но только для sigmoid/tanh);

- использование функций активации ReLU, Leaky ReLU вместо sigmoid/tanh;

- residual connections, как в ResNet;

- gradient clipping при взрыве градиента.

#### 2. Переобучение

Методы решения:

- увеличение обучающей выборки;

- аугментация данных;

- регуляризация L1/L2;

- Dropout;

- Early Stopping;

- уменьшение сложности модели.

#### 3. Недообучение

Методы решения:

- увеличить размер модели;

- обучать дольше;

- подобрать learning rate;

- использовать более подходящую архитектуру;

- улучшить признаки или качество данных;

- уменьшить слишком сильную регуляризацию.

#### 4. Большие вычислительные затраты

Методы решения:

- использование GPU/TPU;

- mini-batch обучение;

- квантизация;

- использование предобученных моделей.

#### 5. Необходимость большого объема данных

Методы решения:

- pre-training + fine-tuning;

- data augmentation;

- synthetic data.
