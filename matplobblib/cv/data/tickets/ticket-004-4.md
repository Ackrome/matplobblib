# 4

4. Представление цифровых изображений: пиксели, разрешение, цветовые модели.

Математически изображение можно описать как непрерывную двумерную функцию яркости f(x, y), которая при оцифровке дискретизируется и преобразуется в матрицу/тензор.

Пиксель - каждый элемент этой матрицы/тензора, представляется в виде целого числа, обычно в диапазоне от 0 до 255.

Существует также вероятностный подход к анализу пикселей. В рамках этого подхода изображение можно представить через распределение вероятностей яркости пикселей (где оценивается число градаций яркости). При этом математическое ожидание яркости определяет среднюю яркость всего изображения, а дисперсия служит мерой его контраста.

Цветовые модели:

- Для кодирования черно-белых изображений используют градации только одного цвета (оттенки серого). Они представляются в виде двумерной матрицы интенсивностей пикселей разрешения H x W, где H — высота, а W — ширина изображения.

- Для кодирования цветных изображений чаще всего используется сочетание трех базисных цветов: красного (R), зелёного (G) и синего (B). Цветное изображение разрешения H x W представляется в виде трехмерного тензора размерности 3 x H x W (или H x W x 3), который объединяет интенсивности базисных цветов для каждого пикселя.
